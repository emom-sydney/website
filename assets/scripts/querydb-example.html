DynamoDB Query Example

/** Set up Amazon S3 and DynamoDB Ensure you have an S3 bucket set up to host your static website. Create a DynamoDB table with the data you want to query. [1] Create an IAM Role for Browser-based Access Navigate to the IAM console (https://console.aws.amazon.com/iam/ ). Create a new role for Web Identity Federation. Attach a policy that allows querying your specific DynamoDB table. Set up Amazon Cognito Go to the Amazon Cognito console (https://console.aws.amazon.com/cognito/ ). Create a new Identity Pool. Configure it to use your IAM role for unauthenticated identities. Create the HTML File Create an HTML file with a form for user input and a submit button. Add a div to display the query results. Add JavaScript to Query DynamoDB Include the AWS SDK for JavaScript in your HTML file. Write a function to query DynamoDB using the SDK. [2] Update the page with the query results. Disable the submit button if the result count exceeds 7. Upload to S3 and Configure for Static Website Hosting Upload your HTML file to your S3 bucket. Enable static website hosting for your bucket. Test and Verify Access your S3-hosted website and test the functionality. **/
